# مصحف القارئ محمد أيوب

تطبيق ويب بسيط يعرض سور وآيات القرآن الكريم بالاعتماد على واجهات Qur'an.com و Islamic.network مع دعم البحث، التلاوة، الحفظ، وتخصيص المظهر.

## المزايا الرئيسية
- تصفح قائمة السور مع تصفية حسب مكان النزول والبحث الفوري.
- استعراض الآيات مع تلاوة صوتية للشيخ محمد أيوب وإمكانية المتابعة التلقائية.
- ترجمة قابلة للاختيار، وتغيير حجم الخط، وأنماط ألوان متعددة.
- بحث شامل في المصحف، آية عشوائية يومية، وإشارات مرجعية قابلة للتصدير.
- حفظ الإعدادات محليًا بما في ذلك السجل وحجم الخط والسمات.

## طريقة التشغيل
1. افتح ملف `index.html` في أي متصفح حديث (يتطلب اتصالاً بالإنترنت للوصول إلى واجهات البرمجة والمحتوى الصوتي).
2. استخدم لوح التحكم الجانبي لضبط السمة، الترجمة، وحجم الخط.
3. اختر سورة للاستماع أو القراءة، وابحث داخلها أو في كامل المصحف.

> **ملاحظة:** بعض ميزات المتصفح (مثل النسخ إلى الحافظة أو تشغيل الصوت تلقائيًا) قد تتطلب تفاعلًا مباشرًا أو منح صلاحيات للصفحة.
